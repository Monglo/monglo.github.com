Features

  • MongoDB compatible queries
  • Persistence for Collections
  • Database Call Middleware

Usage

$ git clone https://github.com/Monglo/MongloDB.git
$ cd MongloDB
$ npm install -d
$ make clean &&  make

Builds will be located in main directory folder
